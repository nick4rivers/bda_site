Welcome

The use of structures designed to mimmic the form and function of natural beaver dams is increasingly being applied to the restoration of wetland environments throughout Oregon. Often referred to as Beaver Dam Analogs, these structures take a variety of forms, and can be used to encourage a variety of biological, hydrologic, and geomorphic processes as part of riparian and channel enhancement projects.

BDA Explorer

The BDA Explorer site was set up to encourage communication among restoration practitioners using beaver dam analog (BDA) type restoration techniques. The site allows users to enter basic project information making it available to visitors of the site. As more projects are added the site will act as a database of beaver based restoration projects in OR, and will ideally be used to track project outcomes

Using the Site

Below you will find answers to frequently asked questions related to the use of the BDA Explorer website. However, if you have additional questions please don't hesitate to send us a message.

Frequently Asked Questions

Do I need to create a user account?

Not necessarily. Any user can browse the site and view project information. You only need to create a user account if you are interested in adding your project to the website.

I'm having problems with my user account?

If you are having trouble with yuor username or password just send us a message with your name and email and the site adminstrator will follow up shortly.

How do I enter project locations coordinates?

Project coordinates should be entered in decimal degrees, and ideally include up to six decimal places of accuracy. i.e. 44.123456, -120.123456.
